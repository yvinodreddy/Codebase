# Laptop Comparison Command

This directory contains the `/laptop-comparison` slash command for Claude Code.

## Command File

**laptop-comparison.md** - Complete reusable command for generating comprehensive, mobile-responsive laptop comparison guides

## How to Use

1. Navigate to the BuyComputerLaptop directory:
   ```bash
   cd /home/user01/claude-test/BuyComputerLaptop
   ```

2. Run the command:
   ```bash
   /laptop-comparison
   ```

3. Provide your laptop requirements:
   ```
   I'm looking at these laptops for AI/ML:
   1. Model 1
   2. Model 2
   ...

   Budget: ₹X lakhs
   Requirements: [your specs]
   ```

4. Get instant comprehensive HTML guide with:
   - Detailed performance scores
   - Budget analysis
   - Mobile-responsive design
   - Purchase links for India
   - Complete recommendations
