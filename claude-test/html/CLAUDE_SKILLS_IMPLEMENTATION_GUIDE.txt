SECTION 1: UNDERSTANDING CLAUDE SKILLS
================================================================================

What Are Skills?
--------------------------------------------------------------------------------
Skills = Reusable instruction packages that teach Claude specific ways of working

Think of it like this:
- WITHOUT Skills: You explain your process every single time
- WITH Skills: You explain once, Claude remembers and reuses automatically

Your Insurance Analysis Example:
  BEFORE: 4-5 hours of manual quote comparison, cost calculation, formatting
  AFTER:  15-30 minutes with automatic processing via the skill


How Skills Work (Technical):
--------------------------------------------------------------------------------
1. METADATA SCANNING (Token-Efficient)
   - Claude scans YAML frontmatter of all available skills (~50 tokens each)
   - Reads: name, description, version
   - Does NOT load full content yet

2. RELEVANCE DETECTION (Automatic)
   - User mentions keywords: "insurance comparison", "cost analysis", etc.
   - Claude matches against skill descriptions
   - Decides which skill(s) are relevant

3. PROGRESSIVE DISCLOSURE (Smart Loading)
   - Only loads FULL skill content when needed
   - Keeps conversations efficient
   - Multiple skills can be combined

4. EXECUTION (Sandboxed)
   - Runs in isolated environment
   - Access to code execution, file creation
   - Your data stays private
