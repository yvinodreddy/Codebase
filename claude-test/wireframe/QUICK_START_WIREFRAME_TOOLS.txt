üéì 5-MINUTE FIGMA CRASH COURSE
================================================================================

1. CREATE FRAME
   - Press 'F' or click Frame tool
   - Select "Desktop" (1920x1080)

2. ADD RECTANGLES (for UI elements)
   - Press 'R' or click Rectangle tool
   - Click and drag to create shape
   - Change color in right sidebar

3. ADD TEXT
   - Press 'T' or click Text tool
   - Click and type
   - Change font, size in right sidebar

4. CREATE BUTTON
   - Draw rectangle
   - Add text on top
   - Select both (Shift+click)
   - Right-click ‚Üí "Frame selection" = Button component

5. REUSE COMPONENT
   - Copy button (Cmd/Ctrl + C)
   - Paste (Cmd/Ctrl + V)
   - Changes to original update all copies

6. SHARE
   - Click "Share" button
   - Set to "Anyone with link can view"
   - Copy link ‚Üí send to team
