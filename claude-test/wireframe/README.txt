üì§ HOW TO SHARE WITH REACT DEVELOPERS
================================================================================

IF USING FIGMA:
---------------
1. Click "Share" button
2. Set to "Anyone with link can view"
3. Copy link
4. Send to developers
5. They use "Inspect" panel for CSS/spacing/colors

Optional: Install Anima plugin for React code export


IF USING EXCALIDRAW OR OTHER TOOLS:
------------------------------------
1. Export as PNG images
2. Create folder with all 12 screens
3. Add document with:
   - Color codes (#2196F3 primary blue, etc.)
   - Font specs (Inter 14px, etc.)
   - Spacing info (8px grid)
4. Share folder with team


üí° KEY INFORMATION FOR REACT DEVELOPERS
================================================================================

Colors (Healthcare theme):
--------------------------
Primary:   #2196F3 (blue - trust)
Success:   #4CAF50 (green - healthy)
Warning:   #FF9800 (orange - alert)
Error:     #F44336 (red - critical)
Background: #FAFAFA (light gray)

Typography:
-----------
Font: Inter or Roboto (Google Fonts)
H1: 32px bold
H2: 24px bold
Body: 14px regular
Small: 12px regular

Spacing:
--------
Use multiples of 8: 8px, 16px, 24px, 32px, 40px, 48px

React Component Libraries:
---------------------------
- Material-UI: https://mui.com
- Ant Design: https://ant.design
- Chakra UI: https://chakra-ui.com
